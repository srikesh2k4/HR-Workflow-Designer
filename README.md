# HR Workflow Designer – Fullstack Example

This ZIP contains:

- `frontend/` – React + Vite + React Flow workflow designer
- `backend/` – Node + Express API for automations and workflow simulation

## Quick start

### 1. Install frontend

```bash
cd frontend
npm install
npm run dev
```

The app runs on http://localhost:5173

### 2. Install backend

```bash
cd backend
npm install
npm run dev
```

The API runs on http://localhost:4000
